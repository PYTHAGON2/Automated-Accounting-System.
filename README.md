# Automated Accounting System (AutoSys)

AutoSys is a comprehensive personal finance management and accounting system. It provides users with a powerful dashboard to track income, expenses, and net balance, while offering administrators a complete system overview and monitoring capabilities.

## Features

- **User Dashboard**: Track personal finances, view transaction history, and manage income/expenses.
- **Admin Portal**: System-wide monitoring, user management, and transaction filtering.
- **Responsive Design**: Modern, clean interface that works across devices.
- **Transaction Management**: Easily add, categorize, and filter transactions.

## Technologies Used

- **Flutter**: For the cross-platform application logic and UI.
- **HTML/CSS/JS**: Core web implementation for the accounting dashboard.
- **Provider**: State management for the Flutter components.

## Getting Started

To run the project locally:

1.  Ensure you have Flutter installed.
2.  Clone this repository.
3.  Run `flutter run` to start the application.

For the web version, you can open `index.html` directly in a modern web browser.
